You: How do I scan a network for live hosts and then check for open ports?
Maxim AI:
Here's a two-step approach:

Step 1: Discover live hosts
nmap -sn 192.168.1.0/24 -oN live_hosts.txt

Step 2: Port scan discovered hosts
nmap -sV -sC -iL live_hosts.txt -oA full_scan

This will first ping sweep the /24 subnet, then run version detection + default scripts on all live hosts. Results saved to files for later analysis.

Alternatives: masscan for faster scanning, netdiscover for ARP-based discovery on local networks.
You: What's the difference between aircrack-ng and wifite?
Maxim AI:
aircrack-ng is a manual suite — you control each step (monitor mode, capture, deauth, crack). More flexible, more control.

wifite automates the entire process. It handles monitor mode, scans, attacks WPA/WPS, and cracks — all in one command.

Use aircrack-ng when you need precision. Use wifite for quick automated audits.
